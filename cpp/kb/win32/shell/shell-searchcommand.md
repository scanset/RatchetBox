# Shell.SearchCommand method

Displays the Apps Search pane.

## Syntax


```JScript
Shell.SearchCommand()
```


```VB

Shell.SearchCommand()
```





## Parameters

This method has no parameters.

### JScript

This method does not return a value.

### VB

This method does not return a value.
