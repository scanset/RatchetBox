# ID3DX12PipelineParserCallbacks::NodemaskCb method

Calls the nodemask subobject callback of an object that implements this interface.

## Syntax


```C++
void NodemaskCb(
   
        UINT
       Nodemask
);
```



## Parameters

<dl> <dt>

*Nodemask* 
</dt> <dd>

Type: **UINT**

Details of the nodemask subobject parsed from a pipeline state stream.

</dd> </dl>

## Return value

Returns nothing.
