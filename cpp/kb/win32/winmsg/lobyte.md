# LOBYTE macro

Retrieves the low-order byte from the specified value.

## Syntax

``` c++
BYTE LOBYTE(
   WORD w
);
```

## -parameters

### -param w

The value to be converted.

## -return

Type: **BYTE**

The value to be converted.

|---------|---------|

|Minimum supported server     | Windows 2000 Server [desktop apps only]        |
