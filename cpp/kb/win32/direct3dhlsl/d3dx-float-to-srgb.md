# D3DX\_FLOAT\_to\_SRGB function

Converts a FLOAT value to an SRGB.

## Syntax

``` syntax
FLOAT D3DX_FLOAT_to_SRGB(
   FLOAT val
);
```

## Parameters

<dl> <dt>

*val* 
</dt> <dd>

FLOAT value to convert.

</dd> </dl>

## Return value

The converted SRGB value.
