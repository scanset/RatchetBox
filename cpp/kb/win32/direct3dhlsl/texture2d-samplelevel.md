# Texture2D::SampleLevel methods

Samples a [**Texture2D**](sm5-object-texture2d.md) on the specified mipmap level.

### Overload list



| Method                                                                             | Description                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)          | Samples a texture on the specified mipmap level.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](samplelevel-s-float-float-int-uint-.md) | Samples a texture on the specified mipmap level and returns status about the operation.<br/> |
