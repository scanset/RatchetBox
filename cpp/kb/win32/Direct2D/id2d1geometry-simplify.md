# ID2D1Geometry::Simplify methods

Creates a simplified version of the geometry that contains only lines and (optionally) cubic Bezier curves and writes the result to an [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink).
