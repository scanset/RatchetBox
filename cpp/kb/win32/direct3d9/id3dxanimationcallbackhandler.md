# ID3DXAnimationCallbackHandler interface

An application implements this interface to handle callbacks in animation sets generated by calls to [**ID3DXAnimationController::AdvanceTime**](id3dxanimationcontroller--advancetime.md).

## Members

The **ID3DXAnimationCallbackHandler** interface inherits from the [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) interface. **ID3DXAnimationCallbackHandler** also has these types of members:

-   [Methods](#methods)

### Methods

The **ID3DXAnimationCallbackHandler** interface has these methods.



| Method                                                                  | Description                                                                                                                                                                                                                                        |
|:------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**HandleCallback**](id3dxanimationcallbackhandler--handlecallback.md) | The application implements this method. This method is called when a callback occurs for an animation set in one of the tracks during a call to [**ID3DXAnimationController::AdvanceTime**](id3dxanimationcontroller--advancetime.md).<br/> |



 

## Remarks

The LPD3DXANIMATIONCALLBACKHANDLER type is defined as a pointer to this interface.


```
typedef interface ID3DXAnimationCallbackHandler ID3DXAnimationCallbackHandler;
typedef interface ID3DXAnimationCallbackHandler *LPD3DXANIMATIONCALLBACKHANDLER;
```
