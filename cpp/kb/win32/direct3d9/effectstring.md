# EffectString

Legacy template used to assign a value in an effect file. This template has been replaced by the [**EffectParamString**](effectparamstring.md) template.

``` syntax
template EffectString
{
    < D55B097E-BDB6-4c52-B03D-6051C89D0E42 >
    STRING Value;
}
```
