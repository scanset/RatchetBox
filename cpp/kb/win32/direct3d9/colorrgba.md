# ColorRGBA

Defines a color object with an alpha component. This is used for the face color in the material template definition.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
