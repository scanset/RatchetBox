# MF\_STREAM\_SINK\_SUPPORTS\_ROTATION attribute

Indicates whether the stream sink supports video rotation.

## Data type

**BOOL** stored as **UINT32**

## Requirements

|-------------------------------------|------------------------------------------------------------------------------------|
| Minimum supported client<br/> | Windows 8 \[desktop apps \| UWP apps\]<br/>                                  |
