# CODECAPI\_AVEncAdaptiveMode property

Specifies the adaptive encoding mode for a video encoder.

## Data type

**ULONG** (VT\_UI4)

## Property GUID

**CODECAPI\_AVEncAdaptiveMode**

## Property value

The value of this property is a bitwise **OR** of values from the [**eAVEncAdaptiveMode**](/windows/win32/api/codecapi/ne-codecapi-eavencadaptivemode) enumeration.
