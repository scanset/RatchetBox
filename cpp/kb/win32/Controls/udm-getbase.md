# UDM\_GETBASE message

Retrieves the current radix base (that is, either base 10 or 16) for an up-down control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

The return value is the current base value.
