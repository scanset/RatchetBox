# EM\_ISIME message

Determine with a rich edit control's current input locale is an East Asian locale.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Not used; must be zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used; must be zero.

</dd> </dl>

## Return value

Returns **TRUE** if it is an East Asian locale. Otherwise, it returns **FALSE**.
