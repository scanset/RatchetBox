# TB\_SETSTYLE message

Sets the style for a toolbar control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>

Value specifying the styles to be set for the control. This value can be a combination of [toolbar control styles](toolbar-control-and-button-styles.md).

</dd> </dl>

## Return value

No return value.
