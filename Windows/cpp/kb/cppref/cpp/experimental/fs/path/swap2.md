void swap( path& lhs, path& rhs );

(filesystem TS)

Exchanges the state of lhs with that of rhs. Effectively calls lhs.swap(rhs).

### Parameters

lhs, rhs

-

paths whose states to swap

### Return value

(none)

### Exceptions

noexcept specification:  noexcept
