### Usage

- constexpr declaration specifier

- constexpr if statement

- lambda-declarator that explicitly specifies the function call to be a constexpr function

(since C++17)

### See also

- if statement: if, else

- consteval if statement: consteval

(since C++23)

- switch statement: case, default (as the declaration of the default case label), switch

- goto statement: goto

- continue statement: continue

- break statement: break

- return statement: return, co_return(since C++20)

- do-while loop and while loop: do, while

- for loop and range-based for loop: for
