# Filled Shapes

*Filled shapes* are geometric forms that are outlined by using the current pen and filled by using the current brush. There are five filled shapes:

-   Ellipse
-   Chord
-   Pie
-   Polygon
-   Rectangle

This overview describes the filled shapes.
