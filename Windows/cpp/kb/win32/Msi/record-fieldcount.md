# Record.FieldCount property

The **FieldCount** property of the [**Record**](record-object.md) object is a read-only property that returns the number of fields in the record. Read access to fields beyond this count returns Null values. Write access fails.

This property is read-only.

## Syntax


```JScript
propVal = Record.FieldCount
```
