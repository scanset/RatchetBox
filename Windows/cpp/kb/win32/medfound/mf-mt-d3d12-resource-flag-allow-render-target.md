# MF\_MT\_D3D12\_RESOURCE\_FLAG\_ALLOW\_RENDER\_TARGET attribute

Indicates whether a render target view can be created for the Direct3D resources in the stream associated with the media type. 

## Data type

**UINT32**

## Remarks

The value of this attribute corresponds to the D3D12_RESOURCE_FLAG_ALLOW_RENDER_TARGET member of the [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) enumeration.
