# ID3DXConstantTable::GetBufferSize method

Gets the buffer size of the constant table.

## Syntax


```C++
DWORD GetBufferSize();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

Returns the size of the buffer, in bytes.
