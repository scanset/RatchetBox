# ID3DXLine::GetPattern method

Gets the line stipple pattern.

## Syntax


```C++
DWORD GetPattern();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

Returns the line stipple pattern: 1 is opaque, 0 is transparent.
