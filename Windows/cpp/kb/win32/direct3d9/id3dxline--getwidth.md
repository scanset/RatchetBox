# ID3DXLine::GetWidth method

Gets the thickness of the line.

## Syntax


```C++
FLOAT GetWidth();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**FLOAT**](../winprog/windows-data-types.md)**

The line thickness.
