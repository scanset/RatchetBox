# ID3DXTextureGutterHelper::GetWidth method

Retrieves the width of the texture, in pixels.

## Syntax


```C++
UINT GetWidth();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**UINT**](../winprog/windows-data-types.md)**

Returns the width of the texture, in pixels.
