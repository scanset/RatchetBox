# Matrix4x4

Defines a 4 x 4 matrix. This is used as a frame transformation matrix.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Where:

-   array float matrix\[16\] - Array of 16 floats.
