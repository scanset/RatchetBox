# EffectParamDWord

Defines an effect DWORD. This template replaced the [**EffectDWord**](effectdword.md) template.

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Where:

-   ParamName - Parameter name.
-   Value - Parameter value.
