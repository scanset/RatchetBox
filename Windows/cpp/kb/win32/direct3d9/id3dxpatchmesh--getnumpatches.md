# ID3DXPatchMesh::GetNumPatches method

Gets the number of patches in the mesh.

## Syntax


```C++
DWORD GetNumPatches();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**DWORD**](../winprog/windows-data-types.md)**

The number of patches.
