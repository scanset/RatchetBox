# ID3DXPRTCompBuffer::GetHeight method

Retrieves the height of the texture, in pixels.

## Syntax


```C++
UINT GetHeight();
```



## Parameters

This method has no parameters.

## Return value

Type: **[**UINT**](../winprog/windows-data-types.md)**

Returns the height of the texture, in pixels.
