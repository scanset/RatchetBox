# TimedFloatKeys

Defines a set of floats and a positive time used in animations.

``` syntax
template TimedFloatKeys 
{ 
    < F406B180-7B3B-11cf-8F52-0040333594A3 > 
    DWORD time; 
    FloatKeys tfkeys; 
} 
```

Where:

-   tfkeys - See [**FloatKeys**](floatkeys.md).
