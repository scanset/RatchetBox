# D3DX\_INT\_to\_FLOAT function

Converts an INT value to FLOAT.

## Syntax

``` syntax
FLOAT D3DX_INT_to_FLOAT(
   INT _V,
   FLOAT _Scale
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The v value.

</dd> <dt>

*\_Scale* 
</dt> <dd>

The scale value.

</dd> </dl>

## Return value

The converted int value.
