# continue Statement

Stop executing the current loop ([do](dx-graphics-hlsl-do.md), [for](dx-graphics-hlsl-for.md), [while](dx-graphics-hlsl-while.md)), update the loop conditions, and begin executing from the top of the loop.

continue;



 

## Parameters

None
