# D3DX\_IsNan function

Determines if the value is a NaN (Not a Number).

## Syntax

``` syntax
bool D3DX_IsNan(
   FLOAT _V
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The specified value.

</dd> </dl>

## Return value

true if a NaN; otherwise false.
