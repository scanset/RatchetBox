# asint function

Interprets the bit pattern of an input value as an integer. For more information about the **asint** intrinsic function, see [**asint (DirectX HLSL)**](dx-graphics-hlsl-asint.md).

## Syntax

``` syntax
int asint(
  in  value
);
```

## Parameters

<dl> <dt>

*value* \[in\]
</dt> <dd>

The input value.

</dd> </dl>

## Return value

Type: **int**

The input interpreted as an integer.
