# D3DX\_FLOAT\_to\_INT function

Converts a FLOAT value to INT.

## Syntax

``` syntax
INT D3DX_FLOAT_to_INT(
   FLOAT _V,
   FLOAT _Scale
);
```

## Parameters

<dl> <dt>

*\_V* 
</dt> <dd>

The v value.

</dd> <dt>

*\_Scale* 
</dt> <dd>

The scale value.

</dd> </dl>

## Return value

The converted FLOAT value
