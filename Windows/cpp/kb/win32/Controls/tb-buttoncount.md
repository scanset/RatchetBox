# TB\_BUTTONCOUNT message

Retrieves a count of the buttons currently in the toolbar.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the count of the buttons.
