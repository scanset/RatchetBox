# RB\_GETPALETTE message

Retrieves the rebar control's current palette.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns an **HPALETTE** that specifies the rebar control's current palette.
