# IPM\_ISBLANK message

Determines if all fields in the IP address control are blank.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns nonzero if all fields are blank, or zero otherwise.
