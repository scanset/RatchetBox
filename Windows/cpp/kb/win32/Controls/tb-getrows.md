# TB\_GETROWS message

Retrieves the number of rows of buttons in a toolbar with the [**TBSTYLE\_WRAPABLE**](toolbar-control-and-button-styles.md) style.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the number of rows.
