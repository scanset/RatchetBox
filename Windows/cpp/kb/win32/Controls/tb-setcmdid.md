# TB\_SETCMDID message

Sets the command identifier of a toolbar button.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Zero-based index of the button whose command identifier is to be set.

</dd> <dt>

*lParam* 
</dt> <dd>

Command identifier.

</dd> </dl>

## Return value

Returns **TRUE** if successful, or **FALSE** otherwise.
