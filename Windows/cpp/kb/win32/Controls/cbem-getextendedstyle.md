# CBEM\_GETEXTENDEDSTYLE message

Gets the extended styles that are in use for a ComboBoxEx control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns a DWORD value that contains the [ComboBoxEx control extended styles](comboboxex-control-extended-styles.md) in use for the control.
