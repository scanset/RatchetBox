# UDM\_SETBUDDY message

Sets the buddy window for an up-down control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Handle to the new buddy window.

</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

The return value is the handle to the previous buddy window.
