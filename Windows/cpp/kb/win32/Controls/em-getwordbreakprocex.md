# EM\_GETWORDBREAKPROCEX message

Retrieves the address of the currently registered extended word-break procedure for a rich edit control.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>

Not used; must be zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Not used; must be zero.

</dd> </dl>

## Return value

The message returns the address of the current procedure.
