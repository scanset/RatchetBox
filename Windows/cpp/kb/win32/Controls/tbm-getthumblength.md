# TBM\_GETTHUMBLENGTH message

Retrieves the length of the slider in a trackbar.

## Parameters

<dl> <dt>

*wParam* 
</dt> <dd>Must be zero.</dd> <dt>

*lParam* 
</dt> <dd>Must be zero.</dd> </dl>

## Return value

Returns the length, in pixels, of the slider.
